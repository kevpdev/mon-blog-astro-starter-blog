# Workflow Patterns

## Standard Validation Commands

```bash
# Quality validation sequence (used in all workflows)
pnpm quality              # TypeScript + Lint + Format check
pnpm typecheck           # TypeScript validation  
pnpm build               # Production build test
pnpm test                # Run test suite
```

## Project Health Check

```bash
# Environment validation
echo "Node: $(node --version)"
echo "PNPM: $(pnpm --version)"
echo "Branch: $(git branch --show-current)"

# Project status
git status --porcelain | wc -l
pnpm outdated | head -5
```

## Error Handling Patterns

### Validation with Fallbacks
```bash
# Command with fallback message
pnpm test || echo "No tests found"
pnpm typecheck || echo "TypeScript errors found"
```

### Conditional Execution
```bash
# Only run if file exists
if [ -f "package.json" ]; then
  pnpm install
fi

# Only run if command succeeds
pnpm build && pnpm preview
```
