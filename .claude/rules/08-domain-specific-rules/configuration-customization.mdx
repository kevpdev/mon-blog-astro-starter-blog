## Core Customization Points

### Site Configuration
- Update `src/config/site.ts` with your site details
- Configure social links, author info, SEO defaults
- Set feature flags (darkMode, comments, analytics)

### Astro Configuration  
- Use canonical config from `.claude/snippets/configs/astro.config.mjs`
- Choose variant: standard, production, or image-optimized
- Customize integrations, build settings, Vite options

### Styling System
- Modify design tokens in `src/styles/design-tokens.css`
- Use mobile-first responsive approach
- Follow component-based architecture

### Content Structure
- Configure content collections in `src/content/config.ts`
- Customize blog post schema (title, description, pubDate, tags)
- Set up static page collections
