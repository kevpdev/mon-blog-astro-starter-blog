Ce thème est livré avec l'intégration [@astrojs/mdx](https://docs.astro.build/en/guides/integrations-guide/mdx/) installée et configurée dans votre fichier de configuration `astro.config.mjs`. Si vous préférez ne pas utiliser MDX, vous pouvez désactiver le support en supprimant l'intégration de votre fichier de configuration.

## Pourquoi MDX ?

MDX est une variante spéciale de Markdown qui prend en charge la syntaxe JavaScript et JSX intégrée. Cela déverrouille la possibilité de [mélanger JavaScript et composants UI dans votre contenu Markdown](https://docs.astro.build/en/guides/markdown-content/#mdx-features) pour des éléments comme des graphiques interactifs ou des alertes.

Si vous avez du contenu existant rédigé en MDX, cette intégration devrait faciliter la migration vers Astro.

Voici comment importer et utiliser un composant UI à l'intérieur de MDX.

import HeaderLink from '../../components/HeaderLink.astro';

	Embedded component in MDX
</HeaderLink>
